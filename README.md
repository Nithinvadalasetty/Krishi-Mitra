
## Krishi Mitra – AI-Driven Crop Disease Advisory System

### Project Objective

To assist farmers in identifying crop diseases accurately and provide expert-level remedies and prevention guidance.

### System Architecture

Leaf Image → Crop-wise AI Engine → Health Check → Disease Detection → Advisory Chatbot

### Model Organization

Crop-specific models with health-first diagnosis logic.

### Data Pipeline

Independent train/val/test splits for each crop.

### Why Not a Single Model?

Crop-wise isolation ensures higher accuracy and safety.

### Future Scope

Shared backbone model and full chatbot integration.
